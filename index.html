sinfo

This is a compact graphical system performance monitor for unix-like operating systems based on Dear ImGui.

install

git clone git@g.nosp@m.ithu.nosp@m.b.com:/kaloyansen/imgui.git && cd imgui/sinfo && make

run

./sinfo&
